# 📊 TubeMetrics: YouTube History Analyzer

TubeMetrics helps you visualize your YouTube consumption habits using Google Takeout data and the YouTube Data API.

## ✨ Features
- **Data Export Guide:** Step-by-step instructions for Google Takeout.
- **Visual Analytics:** Breakdown of Shorts vs. Regular videos and category interests.
- **Insights:** Personalized feedback on your viewing patterns.

## 🚀 Setup
1. Clone the repo: `git clone https://github.com/yourusername/TubeMetrics.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file based on `.env.example` and add your **YouTube Data API v3 Key**.
4. Run the app: `python app.py`

## 🛠️ Tech Stack
- **Backend:** Flask, Requests, isodate
- **Frontend:** HTML5, CSS3, Chart.js
